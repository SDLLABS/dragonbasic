STRINGS :: STR$

Converts an integer to a string. 
FUNCTION STR$(
    n
)
Parameters
n - the unsigned integer to convert
Return
A string that is the decimal representation of the number n.
Remarks
The number is converted using base 10, and it is treated as unsigned (no negative values).
