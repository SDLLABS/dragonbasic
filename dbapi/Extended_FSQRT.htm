EXTENDED :: FSQRT#

Calculates the square root value of a fixed-point value. 
FUNCTION FSQRT#(
    f#
)
Parameters
f# - an unsigned, fixed-point value
Return
The square root of f#.
Remarks
Same as SQRT, but more accurate, keeping decimal places to within 1/256.
