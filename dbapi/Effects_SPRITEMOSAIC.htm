EFFECTS :: SPRITEMOSAIC

Toggles whether a sprite should use the mosaic effect or not. 
SUB SPRITEMOSAIC(
    n,
    flag
)
Parameters
n - the sprite (0-127)
flag - 0 to turn off the effect, anything else enables
Return
None.
Remarks
UPDATESPRITES must be called for this to take effect.
