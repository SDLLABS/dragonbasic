RESETTIMER

Resets the current fire count to 0.

Arguments

No arguments

Results

The total number of times the user-defined timer has fired is reset to 0.

Example

; create a timer that fires 20 times a second
maketimer 20
starttimer

; every time 10 fires have occurred, reset the timer
while
    if clocktimer >= 10 then resettimer
loop
