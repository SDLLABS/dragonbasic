DIRECTIVES :: #PALETTE

Import a palette from BMP, PCX or PAL formatted files. 
#PALETTE "file"
Parameters
None.
Return
None.
Remarks
If an image format is used (BMP or PCX), the image must be 16- or 256-colors deep. If a PAL file is used, then any sized palette will do.
See Also
#BITMAP | LOADPAL16 | LOADPAL256
