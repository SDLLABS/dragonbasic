BITMAPPED :: FLIP

Toggles the currently visible buffer in a double-buffered mode. 
SUB FLIP
Parameters
None.
Return
None.
Remarks
In modes 4 and 5, this function will toggle the current backbuffer, displaying the previous one. In mode 3 or tiled modes, this function has no effect. This function should always be used in a VBLANK.
