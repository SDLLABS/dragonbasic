DIRECTIVES :: #CONSTANT

Creates a constant macro for the compiler.
#CONSTANT  {macro}
Parameters
None.
Return
None.
Remarks
Macros can be good to define sprite indexes, or small code sequences for math, etc. Whenever is encountered in your source code, it is replaced (at compile time) with {macro}.
